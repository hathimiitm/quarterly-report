Valuation Example

Present Value (discounted cash flows):

\( \displaystyle PV = \sum_{t=1}^{N} \frac{CF_t}{(1 + r)^t} \)

Example: \(PV = \dfrac{100}{(1+0.08)^1} + \dfrac{100}{(1+0.08)^2} + \dots \)

Explain each variable: CF_t = cashflow at time t; r = discount rate. Use simple numbers if asked.

Sample: Calculating NPV (Python)


def npv(cashflows, discount_rate):
    pv = 0.0
    for t, cf in enumerate(cashflows, start=1):
        pv += cf / (1 + discount_rate) ** t
    return pv

cashflows = [100, 100, 100, 100, 100]
print(npv(cashflows, 0.08))
